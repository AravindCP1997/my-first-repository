Discounted Cash Flow Methods

Discounted cash flow (DCF) methods are the financial valuation methods used to estimate the value of a project or an asset based on its expected future cash flows.

Financial worth of the project or the asset is determined by discounting cash flows arising from the project or the asset.

Commonly used DCF methods include:

Net Present Value

Net Present Value (NPV) of a project or an asset is the difference between present value of cash inflows and present value of cash outflows arising from the project or the asset.

NPV is positive when present value of cash inflows exceeds the present value of cash out flows and is negative when present value of cash outflows exceeds present value of cash inflows.

A project or an asset is financially viable when it's NPV is greater than or equal to zero.

Profitability Index

Profitability Index (PI) of a project or an asset is the ratio between present value of cash inflows and present value of cash outflows arising from the project or the asset.

PI is greater than one, when present value of cash inflows exceeds the present value of cash out flows and is less than one when present value of cash outflows exceeds present value of cash inflows.

A project or an asset is financially viable when it's PI is greater than or equal to one.

Internal Rate of Return

Internal Rate of Return (IRR) of a project or an asset is the rate of return at which the present value of cash inflows arising from the project or the asset equals present value of cash outflows from the project or the asset.

At IRR, NPV of the project or the asset is zero and PI is one.
