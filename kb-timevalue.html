Time Value of Money

Time value of money is the concept which states that a sum of money received today is worth more than an identical sum to be received on a future date. The fact that the money received today can be invested to generate some income by the future date underlies the concept.

Time value of money is the concept with which sum of money or cashflows arising, from an asset or a project, at differents points of time is evaluated. While doing so, it is important to understand Present Value and Future Value.

Present Value and Future Value

Present Value

Present value is the present worth of a sum of money or cashflow arising on a future date. Present value is calculated by discounting the sum of money or cash flow using the expected rate of return. Present value is also the sum of money to be invested today to generate the required cash flow on a future date.

PV = FV ( 1 + r ) n

Where:

  • PV = Present Value
  • FV = Future Value
  • r = Rate of return
  • n = Number of periods

Future Value

Future value is the worth of a sum of money on a future date, if invested on a specified rate of return. Present value is calculated by compounding the sum of money or cash flow using the expected rate of return.

FV = PV × ( 1 + r ) n

Where:

  • FV = Future Value
  • PV = Present Value
  • r = Rate of return
  • n = Number of periods

Annuity and Perpetuity

Annuity

An annuity is a series of equal payments made at regular intervals over a specified period of time. Annuities can be classified as ordinary annuities (payments made at the end of each period) or annuities due (payments made at the beginning of each period).

FV of Annuity = P × ( 1 + r ) n - 1 r

Where:

  • FV = Future Value
  • P = Payment
  • r = Rate of return
  • n = Number of periods or payments

Perpetuity

A perpetuity is a type of annuity that continues indefinitely, with no end date. It provides a constant stream of cash flows for an infinite period of time. The present value of a perpetuity can be calculated using the formula: PV = C / r, where C is the cash flow per period and r is the discount rate.

Key Formulas

  • Present Value (PV) = FV / (1 + r)^n
  • Future Value (FV) = PV * (1 + r)^n
  • Present Value of Annuity = Pmt * [(1 - (1 + r)^-n) / r]
  • Future Value of Annuity = Pmt * [((1 + r)^n - 1) / r]
  • Present Value of Perpetuity = C / r
  • Future Value of Perpetuity = C / r (assuming constant cash flow)
