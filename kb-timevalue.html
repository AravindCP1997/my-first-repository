Time Value of Money

Time value of money is the concept that a sum of money received today is worth more than an identical sum to be received on a future date. The fact that the money received today can be invested to generate some income by the future date substantiates the concept.

Time value of money is the concept with which sum of money or cashflows arising, from an asset or a project, at differents points of time, is evaluated. Present Value (PV) and Future Value (FV) are the measures used to evaluate or compare cash flow arising at different points of time.

Present and Future Value

Present Value

Present value is the present worth of a sum of money or cashflow arising on a future date. Present value is calculated by discounting the sum of money or cash flow using the expected rate of return. Present value is also the sum of money to be invested today to generate the required cash flow on a future date.

PV = FV ( 1 + r ) n

Where:

  • PV = Present Value
  • FV = Future Value
  • r = Rate of return
  • n = Number of periods

Future Value

Future value is the worth of a sum of money on a future date, if invested on a specified rate of return. Present value is calculated by compounding the sum of money or cash flow using the expected rate of return.

FV = PV × ( 1 + r ) n

Where:

  • FV = Future Value
  • PV = Present Value
  • r = Rate of return
  • n = Number of periods

Annuity and Perpetuity

Annuity

An annuity is a series of equal payments made at regular intervals over a specified period of time. Annuities can be classified as ordinary annuities (payments made at the end of each period) or annuities due (payments made at the beginning of each period).

PV of Annuity = P × ( 1 + r ) n - 1 r × ( 1 + r ) n

Where:

  • PV = Present Value
  • P = Payment
  • r = Rate of return
  • n = Number of periods or payments
FV of Annuity = P × ( 1 + r ) n - 1 r

Where:

  • FV = Future Value
  • P = Payment
  • r = Rate of return
  • n = Number of periods or payments

Perpetuity

A perpetuity is a type of annuity that continues indefinitely, with no end date. It provides a constant stream of cash flows for an infinite period of time.

PV of Perpetuity = P r

Where:

  • PV = Present Value
  • P = Payment
  • r = Rate of return
