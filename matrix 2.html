Matrix

Matrix is an application that reads text-content from JSON (.json) files. User can create query of any depth/level on the data and get output in a matrix structure (rows × columns). User can also filter data at any depth/level.

How to:

  1. Open and Close a File

    Navigate File → Open, to open files. Or user can drag and drop files to the display to open. Files once open is saved in the local storage until user makes a close command (File → Close).

  2. Create a Query

    Click Query button. Query Box will pop up. Each query has two parts. Query Title and Query Path. User can give any Query Title of their choice. Query Path will determine which data to fetch. Query Path is case specific (e.g.: for fetching First Line of Address, use Query Path: 'Address/First Line'). User can add queries on clicking the Add Query button.

  3. Apply a Filter

    Click Filter button. Filter Box will pop up. Each Filter has two parts. Filter Path and Filter Value. Filter first sends query through the Filter Path given. If the response of the query is not the Filter Value specified by the user, that object will be filtered out. Filter Path and Filter Value are case specific. User can add filters on clicking the Add Filter button.

  4. Save Matrix

    Navigate File for Save options. Matrix can be saved either as CSV or as JSON.

  5. Shortcuts
    1. F5: Refresh.
    2. F11: Toggle Full-Screen.
